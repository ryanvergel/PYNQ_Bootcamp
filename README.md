# PYNQ Bootcamp

PYNQ Bootcamp introduces students to artificial intelligence, machine learning, computer vision, Python, and hardware-accelerated applications using the AMD Kria KV260 Vision AI Starter Kit and Jupyter Notebooks.

## Links

- [PYNQ Bootcamp website](https://xilinx.github.io/PYNQ_Bootcamp/)
- [Training modules](https://xilinx.github.io/PYNQ_Bootcamp/Trainings.html)
- [PYNQ project](https://www.pynq.io/)

## Repository contents

The `master` branch contains the bootcamp Jupyter Notebooks, supporting models, drivers, and hackathon materials. The `website` branch contains the published PYNQ Bootcamp website and read-only HTML versions of the training modules.

The 2026 advanced training modules include computer vision, intermediate Python, pose detection, board-to-board communication, prompt engineering, and robot simulation.
